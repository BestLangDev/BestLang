# BestLang

> PHP 是世界上最好的语言！

## 目的
为了避免 Web 课程成为水课，某人决定 ~~成为偶像~~ 再造一个 PHP 框架。

## 项目计划
- [x] 基本 Controller 支持
- [x] 请求相关方法 (Post / Session / Cookies ...)
- [x] 常用数据库 (MySQL / SQLite) 支持
- [x] 基本 Model 支持
- [x] 常用缓存 (Redis / Memcached / WinCache) 支持
- [x] Token 认证支持
- [x] Composer 支持
- [x] 404 / 500 提示页
- [ ] 自定义 URL 路由
- [ ] 基本 View 层支持 (集成第三方模板引擎)
- [ ] 日志支持
- [ ] 高级 Model 支持 (关联 / 映射 / 聚合 ...)
